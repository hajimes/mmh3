mmh3 documentation
==================
mmh3 is a Python extension for
`MurmurHash (MurmurHash3) <https://en.wikipedia.org/wiki/MurmurHash>`_,
a set of fast and robust non-cryptographic hash functions invented by Austin
Appleby.
